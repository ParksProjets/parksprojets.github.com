GSJS( options )

Le constructeur de GSJS

Paramètre

options - Objet, pouvant avoir les attributs suivant:
container - Element du DOM où GSJS va s'afficher
ortho - Boolean, Active la projection orthogonale. Par défaut: false
debug - Boolean, Afficher le compteur de FPS. Par défaut: false
autoRun - Boolean, Démarrer GSJS immédiatement après le constructeur. Par défaut: false
useWebGL - Boolean, Forcer GSJS à utiliser le WebGL, si possible. Par défaut: true

GSJS.createDemiDroite( a, b [, color ] )

Créé la demi-droite [a,b)

Paramètres

a - Objet, ayant les attributs suivant:

y - Int, position y du point
z - Int, position z du point

y - Int, position y du point

Valeur de retour

Cette fonction retourne this

GSJS.loadObject( objet )

Charge et affiche la figure passéee paramètres.

Paramètre

objet - Objet, ayant les attributs suivants:
vertex - Array, Coordonnées des sommets de la figure (Array(x,y,z))
edges - Array, Arrètes de la figure (Array(index de vertex, index de vertex))
faces - Array, Faces de la figure (Array(index de vertex, ...))
names - Array, optionnelle, Nom des sommets, indexés par rapport à vertex

Valeur de retour

Cette fonction retourne this

Exemple

var Cube = {
	"vertex": [ [100,100,100], [100,-100,100], [ -100, -100, 100 ], [ -100, 100, 100 ], [ 100, 100, -100 ], [ 100, -100, -100 ], [ -100, -100, -100 ], [ -100, 100, -100 ] ],
	"edges": [ [0,1], [1,2], [2,3], [3,0], [4,5], [5,6], [6,7], [7,4], [0,4], [1,5], [2,6], [3,7] ],
	"faces": [ [0,1,2,3], [0,4,5,1], [4,5,6,7], [2,3,7,6], [1,2,6,5], [0,3,7,4] ],
	"names": ["A", "B", "C", "D", "E", "F", "G", "H"]
};

GSJS.Interface.addControlButton( type, element )

Ajoute element en tant que boutton de type.
Dès qu'un boutton est cliqué, la classe "active" s'ajoute sur les autres bouttons du même type.

Paramètres

type - String, type de boutton. Peut être:
"addPoint", boutton pour ajouter un point
"addDroite", boutton pour ajouter une droite
"addSegment", boutton pour ajouter

element - element du DOM représentant le boutton

Valeur de retour

Cette fonction retourne this
